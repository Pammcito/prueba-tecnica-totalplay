# Prueba Técnica - Landing Totalplay

## Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Swiper.js

## Instalación

Clonar repositorio:

```bash
git clone https://github.com/Pammcito/prueba-tecnica-totalplay.git
```

## Ejecución local

Abrir el archivo `index.html`

o utilizar Live Server en VS Code.

## Build

No requiere proceso de compilación.
